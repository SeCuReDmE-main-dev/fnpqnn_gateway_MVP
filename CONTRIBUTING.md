# Contributing

Official support is PR-based. Changes must preserve `SCHOOL_TOOL_GOVERNANCE.md`.

Accepted contributions are small, test-backed, and secret-safe. Provider changes must keep Codex/OpenAI and Antigravity/Gemini as the only official school routes.

Do not add Ollama Cloud, uncensored local AI, unknown agent routes, token storage, cookie export, or `.env` mutation as official classroom behavior.
